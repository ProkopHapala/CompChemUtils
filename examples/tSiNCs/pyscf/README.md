# pyscf

PySCF-specific tests under the tSiNCs vibration study — GPU timing and basis-set exploration.

- **test_adamantane.py** — GPU4PySCF single-point on adamantane and Si₁₀H₁₆
- **test_gpu.py** — GPU availability / device smoke test
- **list_basis.py** — enumerate shipped PySCF basis `.dat` files and verify builds
- **pyscf_GPU.md** — GPU4PySCF install and usage notes

Main vibration CLI: [`../vib_spectra.py`](../vib_spectra.py).
