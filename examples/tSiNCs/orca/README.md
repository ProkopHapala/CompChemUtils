# orca

ORCA quantum-chemistry cluster example for large-molecule frequencies (adamantane) — shell wrappers, not integrated with `vib_spectra.py`.

- **adamantane_mpi.inp** — ORCA input for parallel frequency calculation
- **run_orca_mpi.sh** — launch ORCA with MPI on cluster
- **install_openmpi.sh** — OpenMPI build notes for ORCA MPI runs
- **orca.md** — ORCA-specific setup and workflow notes
