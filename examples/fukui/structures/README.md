# structures

Input geometries for Fukui and adsorption-scan workflows in [`../README.md`](../README.md) — Ag clusters, Ag(111) slabs with adatoms, and molecule+metal scan movies (XYZ, POSCAR, CIF).

- **Ag111_*_adatom.*** — 2×2×2 and 3×3×2 Ag(111) slabs with adatom (XYZ, POSCAR, CIF)
- **H2O_Ag*_scan.xyz** / **CH2O_*_scan.xyz** / **maleic_anhydride_*_scan.xyz** — rigid-scan trajectory frames (symmetric vs asymmetric adatom placement, Ag₄ vs Ag₇ vs slab)

Generated locally; large result cubes live under [`../results/`](../results/README.md) (typically not committed).
